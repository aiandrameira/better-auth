## 💻 Projeto

Esse projeto visa mostrar como funciona o NGXS, uma biblioteca de gerenciamento de estado para Angular. Apesar de ser um projeto simples, ele é funcional e pode ser utilizado como base para projetos maiores. O projeto é um gerenciador de produtos, onde é possível adicionar, editar e excluir produtos.

## ✨ Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [Angular](https://angular.io/)
- [NGXS](https://www.ngxs.io/)
- [AiWeb](https://github.com/aiandra-team/ai-web)

---

## 🚀 Instalação

1. Instale as Dependências:

```bash
npm install
```

2. Execute o projeto:

```bash
npm run dev
```

Por fim, a aplicação estará disponível em `http://localhost:4201`
